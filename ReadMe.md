## Technology stack

- Language: [TypeScript v5][5]
- Component engine: [WebCell v3][1]
- Component suite: [BootCell v2][7]
- PWA framework: [Workbox v7][8]
- Package bundler: [Parcel v2][9]
- CI / CD: GitHub [Actions][10] + [Pages][11]

## Development

```shell
npm i pnpm -g
pnpm i
npm start
```

## Deployment

```shell
pnpm build
```
